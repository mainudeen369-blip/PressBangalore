## Project Overview

AP Media is a verified news and media platform focused on **trust, fact-based content, and editorial control** — differentiated from typical social media through admin verification, investigation workflows, and user credibility systems.

A working UI demo has been delivered. This quotation covers **full-stack production development** as agreed in the 24 June 2026 meeting.

---

## Agreed Scope (Summary)

| Area | Deliverables |
|------|--------------|
| **Platform core** | Breaking news, scenes, and user-generated posts with photo, video, audio, and document uploads |
| **User system** | Registration, profiles, roles (reporter, investigator, SME, professional), subscription status, followers, post counters |
| **Editorial** | Admin verification before publish; incline/decline approval flow; counter-statements and evidence support |
| **Reporter module** | Registration, bureau verification, manager approval, digital ID card, status tracking |
| **Investigation (IR)** | Request creation, IR number generation, regional routing, IO assignment, two-step verification, public tracker |
| **Engagement** | Like, dislike, comment, endorse, circulate, sponsor, save, and report actions |
| **Communication** | In-app chat alongside news and posting |
| **Admin portal** | Separate screens for content, reporters, investigations, ads, and role-based access (Super Admin / Admin / Sub Admin) |
| **E-Paper & Live TV** | Publisher subscriptions via wallet; live channel streaming (URL / YouTube); RSS syndication; publisher dashboard |
| **Wallet** | Balance for e-papers, creator sponsorship, and investigation requests (min. ₹1,000 balance) |
| **Hub** | Products, services, courses, marketplace, jobs, organizations, events, and community groups |
| **Trust system** | Fact scores, trust scores, evidence counts — AI-assisted scoring (algorithm design + integration points) |
| **Vira (AI avatar)** | Architecture and integration framework for user-authorized digital clone (chat/call/debate with consent) |
| **Security** | CCTV feed linking, surveillance requests, evidence storage hooks |
| **Technical** | React web app · Python API · scalable PostgreSQL DB architecture · real-time data demo |
